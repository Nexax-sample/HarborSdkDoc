# [NexaX] Harbor Sdk Doc
Harbor SDK is developed and maintained by NexaX Co.

## Releases
Current latest version: **`1.0.2`**

## Features

- Easy integration with Google AdMob
- Distribute ads in both first flow and in-app experience.
- Support for multiple ad formats (Banner, Interstitial, Native)
- Unified and type-safe API
- Lifecycle-aware ad handling
- Centralized ad loading, caching, and displaying

## Requirements

- Android SDK 24 (Android 7.0, Nougat) or higher
- Java 8 or higher
